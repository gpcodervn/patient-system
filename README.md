# Patient Registration System

This a simple Patient system for managing Patient records using JDK 11, Spring Boot and React.

## What did I apply?

- Backend: Spring Boot, Spring JPA, H2, JDK 11
- Frontend: ReactJS, Bootstrap, PrimeReact
